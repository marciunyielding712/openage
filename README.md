## 🛠️ Troubleshooting

### The app does not open

- Right-click the file and choose Run as administrator
- Make sure Windows SmartScreen is not blocking the file
- Download the file again if it looks broken

### Windows says the file is unsafe

- Check that you downloaded it from the GitHub link above
- Choose More info, then Run anyway if you trust the source

### The app opens, but no data loads

- Check the file type
- Try a new file path with simple folder names
- Make sure the file is not open in another program

### The run stops early

- Close extra apps to free memory
- Use a smaller data file
- Check the log view for the line that caused the stop
